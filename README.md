# TrustedBot ✅

**TrustedBot** is a reasoning-based chatbot that leverages **GPT-3.5 Turbo** and the **CoT-SC (Chain-of-Thought Self-Consistency) method** to provide trusted and well-reasoned answers.

---

## Features & Technologies

* Generates **5 reasoning paths** for each question and selects the most consistent answer.
* Chat UI built with **Panel**.
* Uses **OpenAI GPT-3.5 Turbo** for responses.
* Supports **resetting memory**.

---

## Installation

1. Clone the repository:

```bash
git clone https://github.com/MaharZeyad/TrustedBot
cd TrustedBot
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Set your OpenAI API key in a `.env` file:

```
OPENAI_API_KEY=your_openai_key
```

---

## Usage

Run the chatbot:

```bash
python app.py
```

* Type your message in the input box.
* Click **Send** to get answers.
* Click **Reset Memory** to clear the chat history.
